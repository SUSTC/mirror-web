## Grafana 镜像帮助

### Debian / Ubuntu 用户

首先信任 [Package Cloud](https://packagecloud.io/grafana) 的 GPG 公钥:

```bash
curl https://packagecloud.io/gpg.key | sudo apt-key add -
```

再选择你的 Debian / Ubuntu 版本，文本框中内容写进 `/etc/apt/sources.list.d/grafana.list`
